## Short Description

edeg Prompt Floating Library is a lightweight Edge/Chrome browser extension that helps ChatGPT and AI workflow users manage reusable prompts. It provides local prompt storage, search, category management, editing, deletion, and one-click insertion into active web input boxes.

## Maintainer Statement

I am the primary maintainer of this open-source browser extension. I built and maintain the project to help ChatGPT, Codex, and AI workflow users manage reusable prompts more efficiently across web input boxes.

The project includes prompt storage, search, creation, editing, deletion, category management, responsive floating UI positioning, and one-click insertion. I actively maintain browser compatibility, UI behavior, documentation, packaging, and user-facing improvements.

Planned maintenance work includes import/export backup, browser sync, better cross-site input compatibility, optional account-based sync, and improved prompt sharing workflows.

## Why This Project Matters

Prompt reuse is a common workflow for AI users, creators, operators, and small teams. This extension reduces repetitive copying, helps organize prompt libraries, and makes prompt insertion faster inside ChatGPT and other web applications.
