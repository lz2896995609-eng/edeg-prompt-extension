# Changelog

All notable changes to this project will be documented in this file.

## Unreleased

### Added

- Added contribution guidelines for local setup, testing, and pull requests.
- Added a local testing checklist for browser extension behavior.
- Added README links for development and testing documentation.

## 1.0.0 - 2026-06-03

### Added

- Initial public extension package structure.
- Floating `T` prompt button for active web input boxes.
- Expandable prompt library panel.
- Prompt create, edit, delete, and one-click insert workflows.
- Prompt category management.
- Prompt search by title, content, and category.
- Horizontal mouse-wheel scrolling for category tabs.
- Responsive panel positioning around active input boxes.
- Bottom safe-area handling to reduce taskbar overlap.
- Chinese plain-text usage guide for Windows Notepad users.

### Notes

- Prompt data remains stored locally in the browser extension storage.
- Updating the unpacked extension should not remove existing prompt data.
