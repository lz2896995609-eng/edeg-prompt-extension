edeg 提示词悬浮插件使用说明

一、插件用途

这是一个适用于 Microsoft Edge、Google Chrome，以及其他 Chromium 内核浏览器的提示词悬浮插件。

安装后，在网页输入框附近会显示一个 T 悬浮按钮。点击 T 按钮可以展开提示词库，点击提示词卡片后，提示词会自动填入当前输入框。


二、适用浏览器

1. Microsoft Edge
2. Google Chrome
3. 360 极速浏览器、星愿浏览器、百分浏览器等 Chromium 内核浏览器

说明：Firefox 不能直接使用这个版本，需要单独适配。


三、Edge 浏览器安装方法

1. 解压 edeg-prompt-extension.zip。
2. 打开 Edge 浏览器。
3. 在地址栏输入 edge://extensions/
4. 打开“开发人员模式”。
5. 点击“加载解压缩的扩展”。
6. 选择解压后的 edge-extension 文件夹。
7. 安装完成后，打开 ChatGPT 或其他网页输入框测试。


四、Chrome 浏览器安装方法

1. 解压 edeg-prompt-extension.zip。
2. 打开 Chrome 浏览器。
3. 在地址栏输入 chrome://extensions/
4. 打开“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的 edge-extension 文件夹。


五、其他 Chromium 浏览器安装方法

一般操作相同：

1. 打开浏览器扩展管理页面。
2. 开启开发者模式。
3. 选择“加载解压缩扩展”。
4. 选择 edge-extension 文件夹。


六、基本使用方法

1. 打开网页输入框，例如 ChatGPT 对话框。
2. 输入框附近会出现圆形 T 悬浮按钮。
3. 点击 T 按钮展开提示词库。
4. 点击任意提示词卡片，提示词会自动填入当前输入框。
5. 再次点击 T 或点击右上角 X，可以收起提示词窗口。


七、新增提示词

1. 点击“新增提示词”。
2. 输入标题。
3. 选择分类。
4. 输入提示词内容。
5. 点击“保存”。


八、编辑提示词

1. 找到要修改的提示词。
2. 点击右侧编辑图标。
3. 修改标题、分类或提示词内容。
4. 保存后立即生效。


九、删除提示词

1. 找到要删除的提示词。
2. 点击右侧删除图标。
3. 删除后无法自动恢复，删除前请确认。


十、分类功能

1. 点击“分类管理”。
2. 可以新增分类、修改分类名称、删除分类。
3. 新增或编辑提示词时，可以把提示词放到指定分类。
4. 点击分类标签，可以只查看该分类下的提示词。
5. 分类标签较多时，把鼠标放在分类标签区域，使用鼠标滚轮可以左右滑动浏览。


十一、搜索功能

在搜索框输入关键词，可以搜索：

1. 提示词标题
2. 提示词内容
3. 分类名称

搜索时会自动筛选匹配结果。


十二、数据保存说明

插件数据保存在当前浏览器的扩展本地存储中。

注意：

1. 重新加载扩展不会删除已有提示词。
2. 更新插件文件通常不会删除已有提示词。
3. 如果卸载扩展、清空浏览器扩展数据，提示词可能会丢失。
4. 如果要换电脑使用，建议后续增加“导入 / 导出”功能用于备份和迁移。


十三、常见问题

1. 提示词窗口被任务栏遮住怎么办？

新版已经加入底部安全距离和自动高度收缩。如果仍然遮挡，可以把浏览器窗口稍微上移或缩小一点，也可以把 Windows 任务栏设置为自动隐藏。

2. 提示词窗口位置不对怎么办？

插件会根据当前输入框位置自动调整。如果页面结构特殊，可能需要刷新网页后再试。

3. 为什么换电脑后没有提示词？

当前版本使用浏览器本地存储，不是云同步。换电脑需要后续增加导入 / 导出或账号同步功能。

4. 可以在所有网站使用吗？

大多数普通输入框、文本框、可编辑区域都可以使用。个别网站如果输入框结构特殊，可能需要单独适配。


十四、更新插件方法

1. 下载或替换新版插件文件。
2. 解压新版 edeg-prompt-extension.zip。
3. 打开浏览器扩展管理页面。
4. 找到 edeg 提示词插件。
5. 点击“重新加载”。

正常情况下，更新插件不会清空已有提示词。
