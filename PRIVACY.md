# Privacy

edeg Prompt Floating Library is a local-first browser extension.

## What Data Is Stored

The extension stores user-created prompt titles, prompt content, and categories in browser extension local storage.

## Where Data Is Stored

Prompt data is stored locally by the browser extension. The current public release does not include a backend server, account login, analytics, telemetry, or cloud sync.

## Network Requests

The extension does not intentionally send saved prompts to a remote server.

## User Responsibility

Users should avoid storing passwords, private keys, API keys, financial data, or other sensitive secrets as reusable prompts.

## Future Changes

If cloud sync, account login, or shared prompt libraries are added in the future, this privacy document should be updated before those features are released.
