# Security Policy

## Supported Versions

The latest public release is the supported version.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

Please report security issues through GitHub Issues if the report does not contain private user data or secrets.

If a report includes sensitive data, remove private details before posting. This extension is local-first and does not require users to share saved prompts, cookies, access tokens, or private account data when reporting issues.

## Data Handling

The extension stores prompt data in browser extension local storage. It does not include a backend service, account system, or remote sync in the current public release.

Do not paste private prompts, credentials, API keys, or account information into public issue reports.
