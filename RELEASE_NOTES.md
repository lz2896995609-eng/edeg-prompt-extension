# v1.0.0 - Initial public release

Initial public release of edeg Prompt Floating Library.

## Features

- Floating prompt library for Edge and Chrome
- Prompt create, edit, delete, and one-click insert
- Search by title, content, and category
- Category management
- Horizontal category scrolling
- Responsive floating panel positioning
- Local-first browser storage

## Package

The installable extension package is:

`edeg-prompt-extension.zip`

Upload this zip file as the GitHub Release asset for `v1.0.0`.
